# Images Directory

## Required Images

### og-image.png
- **Size**: 1200x630px
- **Purpose**: Social media preview image (Open Graph, Twitter Cards)
- **Content**: Should show the sunscreen chooser branding and purpose
- **Format**: PNG or JPG
- **Tools**: Canva, Figma, or any image editor

### favicon.ico
- **Size**: 16x16px, 32x32px, 48x48px (multi-size ICO)
- **Purpose**: Browser tab icon
- **Content**: Simple sun icon or logo
- **Tools**: [Favicon Generator](https://favicon.io/)

## Temporary Placeholders

For development, you can use:
- **og-image.png**: Any 1200x630px image
- **favicon.ico**: Browser will use default if missing
